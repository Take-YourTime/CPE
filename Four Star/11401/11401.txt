有長度1 ~ n的棍子，請問可以組成多少個不同的三角形？

==========

令三角形的三個邊為：x、y、z

並且讓x始終為最大的邊，y為第二大

→	x < y + z, by 兩邊和大於第三邊
	
	x - y < z, z < x → x - y < z < x


假設 x = a，y的值可以為1 ~ a - 1。

當y = a-1時，z的值可以為[1+a-(a-1)] ~ a-1 = 2 ~ a-1 → 有a-2個解

→	總共有 0 + 1 + 2 + 3 + ...... + x-2 = (1 + x-2) * (x-1) / 2個解

因為當y > x/2時，會有y = z的情況發生，所以還需要扣掉(x-1) / 2

接著再扣掉所有y, z對調的情況，因此全部需要再除以2

至此我們可以得到公式：
			sum[i] = sum[i-1] + [(1 + x-2) * (x-1) / 2 - (x-1) / 2] / 2, i > 3
			sum[3] = 0

跑迴圈把全部情況算出來

最後輸出即可
