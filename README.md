# CPE Coding Exercise
Coding exercise for CPE (Collegiate Programming Examination)  
The questions are form UVa online judge.

-> My online judge personal page: https://onlinejudge.org/index.php?option=onlinejudge&Itemid=20&page=show_authorstats&userid=1647494  
-> My uHunt: https://uhunt.onlinejudge.org/id/1647494  
  
-> The star classification of each questions: https://par.cse.nsysu.edu.tw/~advprog/star.php

> **Note:** Entries with a trailing `/` are subdirectories containing the solution files. Entries without `/` are problem IDs whose solution files (`<id>.cpp`, `<id>.pdf`, etc.) are stored directly in the parent category folder.

> Each numbered folder (e.g. `10162/`) contains:
> - `<id>.cpp` — C++ solution source code
> - `<id>.pdf` — Problem statement (PDF)
> - `<id>.png` — Screenshot of accepted submission (if available)
> - `<id>.txt` — Additional notes or explanation (if available)
